# Digital Vault API

Ein modernes Backend-Projekt zur sicheren Verwaltung digitaler Notizen. Entwickelt mit **Python** und **FastAPI**, um saubere RESTful-Prinzipien, Datenvalidierung und automatisierte Dokumentation zu demonstrieren.

## Key Features

- **Full CRUD Support**: Erstellen, Lesen und Löschen von Einträgen.
- **Kategorie-Suche**: Gezieltes Filtern von Notizen über Query-Parameter.
- **Auto-Dokumentation**: Integrierte Swagger UI (OpenAPI) für interaktive Tests.
- **Data Validation**: Strikte Typisierung durch Pydantic-Modelle.

## Tech Stack

- **Sprache:** Python 3.x
- **Framework:** FastAPI
- **Web-Server:** Uvicorn
- **Validierung:** Pydantic
